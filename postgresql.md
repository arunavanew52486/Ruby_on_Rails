# PostgreSQL

## installation ubuntu

## initialize
### start postgres sql:
```
sudo -u postgres psql
```

### list of db's:
```
\l
```
or 
```
\list
```
or
```
select datname from pg_database;
```

### select a db:
```
\c db_name
```
or
```
\connect db_name
```

### list of table's in a selected db:
```
\dt
```
